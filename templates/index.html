    Vendor controller tools

    First try to switch hardware RAID controllers to HBA, IT, JBOD, or passthrough mode so DriveProof can see the physical drives directly. Use vendor RAID/HBA tools only when direct passthrough is not possible or when controller-specific diagnostics are required. Downloads are saved to DRVTOOLS automatically in the live system.
