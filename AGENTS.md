# Agent Instructions

## Agent skills

### Issue tracker

Issues are tracked in GitHub Issues for `joreichhardt/driveproof`; external PRs are not a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Triage roles use the default canonical labels: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

This repo uses single-context domain docs: root `CONTEXT.md` plus `docs/adr/`. See `docs/agents/domain.md`.
