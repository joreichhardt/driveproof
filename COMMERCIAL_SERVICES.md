# Commercial Services

DriveProof can be used in commercial settings. The project is well suited to
service-oriented business models around open source distribution.

Typical offerings:

- Custom branding
  - logo, colors, product naming, report branding
  - white-label kiosk UI

- Build service
  - customer-specific NixOS live images
  - hardware-specific package sets
  - preconfigured kiosk defaults

- Integration service
  - USB dock validation
  - SMART passthrough checks
  - server batch-testing workflows

- Support and maintenance
  - bug fixes
  - release management
  - compatibility testing

- Deployment media
  - prepared USB sticks
  - factory setup images
  - internal lab images

Important note:

Selling DriveProof or DriveProof-based media is compatible with open source
distribution, but third-party components in the live image keep their own
licenses and obligations. Commercial packaging does not remove those upstream
license requirements.
